# QuickShow 🎬

QuickShow is a full-stack MERN movie ticket booking platform designed to simulate a modern cinema booking experience. Users can browse currently playing movies, view details, select seats, and book tickets through an interactive interface. The platform also includes an admin dashboard that allows administrators to manage shows, monitor bookings, and track revenue.

The project follows a scalable architecture using the MERN stack (MongoDB, Express.js, React, Node.js) and focuses on clean UI design, modular components, and maintainable code structure.

## Features

* Browse currently playing movies
* View movie details and ratings
* Interactive seat selection layout
* Ticket booking interface
* Admin dashboard for show management
* Add shows with custom date and time
* View show listings and bookings
* Modern responsive UI

## Getting Started

Clone the repository:

```
git clone https://github.com/PrashBytes/QuickShow.git
```

Navigate to the client folder:

```
cd QuickShow/client
```

Install dependencies:

```
npm install
```

Run the development server:

```
npm run dev
```

The application will start on:

```
http://localhost:5173
```
